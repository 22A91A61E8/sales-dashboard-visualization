# Screenshots

This directory contains screenshots of the Sales Dashboard Visualization at different viewport sizes.

## Required Screenshots

Please capture and upload the following screenshots:

### 1. Desktop View (`desktop.png`)
- **Viewport Size**: 1920x1080
- **URL**: https://22a91a61e8.github.io/sales-dashboard-visualization/
- **Instructions**:
  1. Open the dashboard in Chrome/Firefox
  2. Press F12 to open Developer Tools
  3. Click the device toolbar icon (or press Ctrl+Shift+M)
  4. Set dimensions to 1920x1080
  5. Take screenshot showing full dashboard with all components

### 2. Tablet View (`tablet.png`)
- **Viewport Size**: 768x1024
- **Instructions**:
  1. Set browser to 768x1024 using Developer Tools
  2. Capture the responsive 2-column layout
  3. Show how charts adapt to tablet size

### 3. Mobile View (`mobile.png`)
- **Viewport Size**: 375x667
- **Instructions**:
  1. Set browser to 375x667 (iPhone SE size)
  2. Capture the single-column scrollable layout
  3. Show mobile-optimized interface

### 4. Cross-Filter Demo (`cross-filter-demo.png`)
- **Instructions**:
  1. Click on a specific region in the pie chart (e.g., "North")
  2. Capture how all other charts update to show filtered data
  3. Ensure the active filter is visible

### 5. Date Filter Demo (`date-filter-demo.png`)
- **Instructions**:
  1. Select a custom date range using the date pickers
  2. Click Apply
  3. Capture the dashboard with filtered date range visible
  4. Show KPI cards reflecting the filtered period
